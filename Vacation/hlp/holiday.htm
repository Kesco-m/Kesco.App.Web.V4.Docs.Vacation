Заявление на отпуск - приложение для создания и редактирования заявления на отпуск сотрудников.
Просмотр и редактирование заявления на отпуск

Форма предназначена для создания заявления на отпуск сотрудников.
Панель "Замещения по документу" предназначена для создания замещений сотрудников связанных с документом.
Все поля выделенные желтым цветом обязательны для заполнения.

Поля документа:
  • Дата - дата создания документа;
  • От работника - ФИО сотрудника от которого составляется заявление;
  • Организация - название организации сотрудника;
  • Руководителю организации - ФИО руководителя организации, на имя которого составляется заявление;
  • Вид отпуска - вид предоставляемого отпуска;
  • Первый день отпуска - дата начала отпуска;
  • Продолжительность - количество дней отпуска. Праздничные дни не включаются в количество дней отпуска;
  • Последний день отпуска - дата последнего дня отпуска.
  • День выхода на работу - дата первого рабочего дня после отпуска. Отображается только для справки.
  • Сотрудник - ФИО сотрудника, который является заместителем заявителя на время отпуска;
  • Описание документа в архиве - поле для указания дополнительной информации, связанной с документом.

Управляющие элементы:
  • Сохранить - сохраняет документ в БД и создает связанные с ним замещения сотрудников;
  • Обновить - обновляет внешний вид документа и проверяет правильность заполнения всех полей формы;
  • Добавить - добавляет новое замещение сотрудников, на даты документа и связанное с документом;
  • Удалить - удаляет замещение сотрудников. Замещения, которые уже завершились не могут быть удалены. Замещения, которые уже начались, будут сокращены до текущей даты.

Диалог редактирование замещения сотрудников

Диалог предназначен для редактирования замещений сотрудников.

Поля диалога:
  • Сотрудник - ФИО сотрудника, который является заместителем заявителя на указанные даты;
  • с - дата начала действия замещения. Дата начала замещения не может быть установлена ранее текущей даты.
  • по - последний день действия замещения. Дата конца замещения не может быть установлена ранее даты начала замещения или ранее текущей даты. Длительность замещения не может быть более 30 дней.
  • Примечание - дополнительная информация о замещении.

В диалоге расположены следующие управляющие элементы:
  • Сохранить - закрывает диалог с сохранением данных замещения;
  • Удалить - удаляет замещение. Замещения, которые уже завершились не могут быть удалены. Замещения, которые уже начались, будут сокращены до текущей даты.
  • Отмена - закрывает диалог без сохранения данных замещения.
